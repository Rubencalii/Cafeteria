================================================================================
🎛️ CÓMO USAR EL PANEL DE ADMINISTRACIÓN
================================================================================

DESPUÉS DE INICIAR SESIÓN VERÁ 4 SECCIONES:

📊 1. DASHBOARD (Resumen General)
   --------------------------------
   AL ENTRAR VERÁ:
   • Total de reservas recibidas
   • Reservas para hoy
   • Mensajes pendientes de responder
   • Elementos disponibles en el menú

   ¿QUÉ HACER AQUÍ?
   → Revisar diariamente para ver el estado de su negocio
   → Ver si hay reservas nuevas o mensajes urgentes

📅 2. GESTIÓN DE RESERVAS
   ----------------------
   AQUÍ VE TODAS LAS RESERVAS DE SUS CLIENTES:

   INFORMACIÓN DE CADA RESERVA:
   • Nombre del cliente
   • Email y teléfono
   • Fecha y hora solicitada
   • Número de personas
   • Estado: Pendiente/Confirmada/Cancelada

   ACCIONES QUE PUEDE HACER:
   ✅ CONFIRMAR RESERVA → Clic en "Confirmar"
   ❌ CANCELAR RESERVA → Clic en "Cancelar"
   
   FILTROS DISPONIBLES:
   • Ver todas las reservas
   • Solo pendientes (las que necesita revisar)
   • Solo confirmadas (las que ya aceptó)
   • Solo canceladas

   📋 RUTINA DIARIA RECOMENDADA:
   1. Entre cada mañana al panel
   2. Revise reservas pendientes
   3. Confirme las que puede atender
   4. Cancele si no tiene disponibilidad
   5. Llame o envíe email al cliente para confirmar

📧 3. MENSAJES DE CONTACTO
   ------------------------
   AQUÍ VE TODOS LOS MENSAJES DE SUS CLIENTES:

   TIPOS DE MENSAJES QUE RECIBIRÁ:
   • Consultas sobre horarios
   • Preguntas sobre el menú
   • Solicitudes especiales
   • Felicitaciones o quejas
   • Consultas sobre eventos privados

   ESTADOS DE MENSAJES:
   • No leído (nuevos mensajes)
   • Leído (ya los vio)
   • Respondido (ya contestó al cliente)

   ACCIONES:
   ✅ "Marcar Leído" → Cuando haya leído el mensaje
   ✅ "Respondido" → Cuando ya haya contestado al cliente

   📋 RUTINA DIARIA RECOMENDADA:
   1. Revise mensajes no leídos
   2. Responda por email o teléfono al cliente
   3. Marque como "Respondido" una vez que conteste

🍽️ 4. GESTIÓN DEL MENÚ
   --------------------
   AQUÍ CONTROLA QUÉ SE MUESTRA EN SU PÁGINA WEB:

   PARA CADA ELEMENTO DEL MENÚ VE:
   • Nombre del producto
   • Categoría (Café Caliente, Café Frío, Postres, Desayunos)
   • Precio actual
   • Si está disponible o no

   ACCIONES QUE PUEDE HACER:
   ✏️ EDITAR → Cambiar nombre, descripción, precio
   🔄 ACTIVAR/DESACTIVAR → Si se agotó un producto
   ➕ AGREGAR NUEVO → Para productos nuevos
   🗑️ ELIMINAR → Para productos que ya no vende

   💡 CASOS DE USO COMUNES:
   
   ➤ SE AGOTÓ EL TIRAMISÚ:
   1. Busque "Tiramisú" en la tabla
   2. Clic en "Desactivar"
   3. Ya no aparecerá en la página para clientes
   4. Cuando tenga nuevamente, clic en "Activar"

   ➤ SUBIÓ EL PRECIO DEL CAFÉ:
   1. Busque el producto en la tabla
   2. Clic en "Editar"
   3. Cambie el precio
   4. Guarde los cambios
   5. El nuevo precio aparece inmediatamente en la web

   ➤ NUEVO PRODUCTO ESPECIAL:
   1. Clic en "Agregar Elemento"
   2. Llene: Nombre, Descripción, Precio, Categoría
   3. Guarde
   4. Aparece inmediatamente en su página web

⏰ RUTINA DIARIA RECOMENDADA PARA EL DUEÑO
================================================================================

🌅 CADA MAÑANA (15 minutos):
1. Abra su panel: http://localhost:8000/admin.php
2. Revise el Dashboard para ver resumen
3. Vaya a "Reservas" → Confirme reservas del día
4. Vaya a "Mensajes" → Responda mensajes nuevos
5. Si algún producto se agotó → Desactívelo en "Menú"

🌆 CADA TARDE (5 minutos):
1. Revise si hay reservas nuevas para mañana
2. Responda mensajes pendientes
3. Active productos que volvieron a estar disponibles

📅 CADA SEMANA:
1. Revise estadísticas en Dashboard
2. Actualice precios si es necesario
3. Agregue productos nuevos de temporada
4. Revise que toda la información esté actualizada
